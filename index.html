使用说明

🎨 设计规范

统一橄榄绿军事色系，俯视/平面投影视角，符合军用地图标绘规范。所有图标默认朝上为前进方向。

📐 矢量缩放

SVG 格式无损缩放，适用于态势大屏、作战系统、Web 平台及打印输出。可通过 transform: scale() 调整尺寸。

🔄 方向变换

通过 CSS 或 SVG transform="rotate(角度, 中心X, 中心Y)" 实现任意方向标注，适配实时态势展示。

🌓 明暗适配

当前配色适配浅色背景。深色主题下可将填充色反转为 #8aaa8a 系列，描边保持 #c0d0c0。
